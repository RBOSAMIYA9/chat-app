# Chatterbox
Chatterbox is a chat app made using firebase and react js. 

## Features
- Social login (google)
- Creat and delete chat room
- Dark Mode

## Demo 
[ChatterBox](http://chat.ravindrabosamiya.tech/)


## Tools 
- React js 
- Firebase
